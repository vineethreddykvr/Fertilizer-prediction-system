    About Project

    PROBLEM DEFINATION:

    As farmers are good in farming but sometimes unable to know which fertilizer to be used for the crop or land. So this project will predict the fertilizer to be used based on the previous collected data such as temperature, humidity, soil and crop type etc.


    Abstract
    India is a nation of more than 1000 million individuals. Farming is the backbone of Indian economy, contributing around 22 percent of total national output (GDP). The motivation of this project comes from the increasing suicide rates in farmers which may be due to low harvest in crops.Climate and other environmental changes have become a major threat in the agriculture field. Machine learning is an essential approach for achieving practical and effective solutions for this problem. Predicting the fertilizer to be used from the historical available data like temperature, humidity, soil type, crop type and other parameters. We achieved this using the machine learning algorithms, i.e.,KNN, SVM, DecisionTreeClassifier and chosen Random Forest Algorithm which gave an accuracy of 95%. In this project an application is developed which predicts the fertilizer to be used for a particular crop.

    Solution
    Making a model using meachine learning algorithm that predicts the fertilizer to be used for the crop based on the reqirments given to the model based,the output is produced farmers can easily undertand which fertilizer to be used . the output is based on the training of the model based on the previous data.


    About Prediction

    With the aim of sustainable crop production, reduction in the misuse of fertilizers and reduction of input cost by optimizing the present pattern of excessive fertilizer application, our team provides fertilizer recommendation through both the manual and soil test based interpretation of plant nutrients through soil database and static laboratory soil analysis.

    Our team has developed web-based software named Fertilizer Recommendation System(FRS). The system is capable of generating fertilizer recommendations for selected crops by analysing the soil database developed.

    The software requires proportion of nutrients in soil, land type, and crop type and variety information to generate crop-specific instant fertilizer recommendation.
